# Test Reports Directory

This folder stores machine-readable test run artifacts generated by `backend_test.py`.

## Files

- `backend_test_results.json`: latest integration run output (overwritten on each run)
- `backend_test_results_YYYYMMDD_HHMMSS.json`: timestamped historical snapshots

## Usage

- Commit at most one representative report when you need a sample artifact in the repo.
- Avoid committing large volumes of timestamped reports to keep repository history clean.
